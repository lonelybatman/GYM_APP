  Muskelgruppen-Hierarchie – Vollständige Logik  Dokumentation für Claude Code1. GrundprinzipEs gibt 12 wählbare Muskelgruppen. 6 davon sind &apos;Übergruppen&apos; die andere Gruppen vollständig beinhalten. Wenn eine Übergruppe gewählt wird, werden alle enthaltenen Gruppen ausgegraut und können nicht mehr einzeln gewählt werden. Wenn eine enthaltene Gruppe bereits gewählt war, wird sie automatisch abgewählt wenn die Übergruppe gewählt wird.Regel: Eine Gruppe wird ausgegraut sobald EINE der Übergruppen die sie enthält ausgewählt ist — es reicht eine.2. Algorithmus6.1 Auswahl-Logik (Screen 2 — Muskelgruppen wählen)const HIERARCHY = {  Upper:     [&apos;Arms&apos;, &apos;Shoulders&apos;, &apos;Chest&apos;, &apos;Back&apos;, &apos;Core&apos;],  Lower:     [&apos;Legs&apos;, &apos;Glutes&apos;, &apos;Adductors&apos;, &apos;Abductors&apos;, &apos;Calves&apos;],  Pull:      [&apos;Back&apos;, &apos;Core&apos;],  Push:      [&apos;Chest&apos;, &apos;Glutes&apos;],  Anterior:  [&apos;Chest&apos;, &apos;Core&apos;, &apos;Adductors&apos;],  Posterior: [&apos;Back&apos;, &apos;Glutes&apos;, &apos;Abductors&apos;, &apos;Calves&apos;],};// Alle Gruppen die durch aktive Übergruppen abgedeckt sindfunction getCoveredGroups(selected: string[]): string[] {  const covered = new Set&lt;string&gt;();  for (const s of selected) {    if (HIERARCHY[s]) HIERARCHY[s].forEach(g =&gt; covered.add(g));  }  return [...covered];}// Wenn Nutzer eine Gruppe tippt:function onGroupToggle(group: string, selected: string[]): string[] {  const isParent = !!HIERARCHY[group];  const isCovered = getCoveredGroups(selected).includes(group);  if (isCovered) return selected; // ausgegraut → kein Toggle möglich  if (selected.includes(group)) {    // abwählen    return selected.filter(s =&gt; s !== group);  } else {    // hinzufügen: wenn Übergruppe → Kinder aus selected entfernen    let next = [...selected, group];    if (isParent) {      next = next.filter(s =&gt; !HIERARCHY[group].includes(s));    }    // wenn Kind → prüfen ob Übergruppen abgewählt werden müssen    // (nicht nötig: Übergruppe kann Kind nicht mehr enthalten wenn    //  Kind bereits manuell gewählt — Übergruppe bleibt, Kind wird    //  trotzdem ausgegraut. Nutzer muss Übergruppe zuerst abwählen.)    return next;  }}6.2 Detailmuskeln filtern (Screen 3)function getVisibleMuscles(selected: string[], allMuscles: Muscle[]): Muscle[] {  const covered = getCoveredGroups(selected);  const activeGroups = new Set([...selected, ...covered]);  return allMuscles.filter(m =&gt; activeGroups.has(m.muscle_group_name));}
